================================================================
POS INVENTARIO - GUIA RAPIDA DE INSTALACION
================================================================

REQUISITOS
----------
- XAMPP (o cualquier servidor con PHP 7.4+ y MySQL/MariaDB)
- El proyecto copiado en C:\xampp\htdocs\pos\

PASOS
-----

OPCION A - INSTALADOR AUTOMATICO (recomendado)
   1. Iniciar XAMPP y arrancar Apache + MySQL.
   2. Abrir en el navegador:  http://localhost/pos/install.php
   3. Click en "Instalar base de datos".
   4. Listo. Abre http://localhost/pos para entrar.
   5. (Opcional) Borrar install.php por seguridad.

OPCION B - MANUAL
   1. Iniciar XAMPP y arrancar Apache + MySQL.
   2. Importar pos.sql desde phpMyAdmin o consola:
        phpmyadmin -> Importar -> seleccionar pos.sql -> Continuar
        o bien: C:\xampp\mysql\bin\mysql.exe -u root < pos.sql
   3. Si el host de MySQL no es 127.0.0.1:3306 con root sin password,
      editar config.php antes de usar el sistema.

DESPUES DE INSTALAR
-------------------
   Abrir:  http://localhost/pos

CREDENCIALES POR DEFECTO
------------------------
   Administrador:  admin       /  admin
   Vendedor:       vendedor    /  vendedor

REQUISITOS DE PHP
-----------------
   - PHP 7.4 o superior (probado en 8.2)
   - Extensiones habilitadas en php.ini:
       extension=pdo_mysql
       extension=openssl
       extension=gd
   - En XAMPP ya vienen habilitadas por defecto.

PARA OTRA PC
------------
   1. Copiar toda la carpeta pos/ (sin el directorio .git si se prefiere)
   2. Pegarla en C:\xampp\htdocs\ de la otra PC
   3. Iniciar XAMPP (Apache + MySQL)
   4. Ejecutar install.php
   5. Entrar al sistema

NOTAS
-----
- La carpeta del proyecto debe tener permisos de escritura para
  Apache (necesario para subir imagenes de productos y usuarios).
- Si MySQL usa un puerto distinto de 3306, editar DB_PORT en config.php.
- Si el usuario MySQL tiene password, editar DB_PASS en config.php.
- bd-indices.sql ya no es necesario: los indices estan en pos.sql.
